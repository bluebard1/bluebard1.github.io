Simpam - Simple Password Manager

Simpam is a *very* simple password management program for cli. It was propgrammed as a result of my first interactions with C in 2015/16 and allows generating passwords, encrypting, and storing them on the hardrive. It can easily be built from source and taught usage of variables, (standard) types, pointers, file-handling, static memory, functions, and the most common C-libraries. The unsuspensful source code is supplied on Github for "historical" reasons.

Usage generating a password: -g servicename

Encrypted passwords for services are stored on harddrive and simpam -p copies password directly to clipboard


Note that:

* Safe storage of passwords is not granted since this project is rather educational in nature.

* Master passphrase is hard coded into source code -> recompile from source to change

* As of today I am aware that there are infinitely many ways to improve this miniscule software project.
